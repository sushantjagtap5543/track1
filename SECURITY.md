# Security Policy

## Supported Versions

We only support the latest version of GeoSurePath. Please ensure you are running the most recent commit of the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.2.0   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open a public issue. Instead, send an email to security@geosurepath.com.

Our security team will investigate and respond within 48 hours.

### What to include:
- A description of the vulnerability.
- Steps to reproduce (proof of concept).
- Potential impact.
